# 伊利诺伊大学系统董事会成立校长搜索委员会 提名工作已截止

2026年4月6日，伊利诺伊大学系统董事会推进校长遴选工作，已批准成立由23名成员组成的搜索委员会，覆盖八大利益相关群体，包括教师和学生代表。现任校长蒂莫西·基林已宣布将在2027年6月30日合同到期后卸任。

根据伊利诺伊大学系统媒体关系高级作家兼助理总监史蒂夫·维特默介绍，董事会收到提名后，将投票决定委员会成员。此次提名程序与过去两次校长搜索类似，但委员会规模比上次扩大了四人。搜索委员会构成包括：三名校董成员，由董事会主席杰西·鲁伊斯确定人选；八名教师成员，厄巴纳-香槟、芝加哥、春田三个校区分别派出三名、三名、两名代表，各校区参议院产生候选人名单后提交大学参议院会议，再转交董事会；三名学生成员，每个校区一名，由各校区学生会在学生董事参与下遴选学生候选人，董事会要求考虑本科、研究生和专业学生参与；一名学术专业职员代表，由各校区专业咨询委员会产生名单提交系统专业人员咨询委员会后转董事会；一名公务员系统职员代表，由各校区员工咨询委员会产生名单提交系统员工咨询委员会后转董事会；三名行政官员，比之前搜索增加两名，董事会在咨询现任校长和三校区校长后遴选；三名校友协会代表，每个校区一名，由各校区校友协会董事会遴选产生；伊利诺伊大学基金会一名代表，由基金会董事会遴选产生名单提交董事会。

现任校长基林2012年接替上任后，已经任职超过11年，任期长度超过过去四任校长任期总和。过去二十年间，校长职位争议不断，前两任校长分别因录取丑闻和教师对组织举措的担忧而辞职。

根据时间安排，搜索委员会提名已于4月8日截止。董事会随后将确定委员会成员，启动下一阶段校长遴选工作。
