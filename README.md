## ✅ 核心原则

1. **严格按原文编译，绝不编造** - 所有信息均有原文依据
2. **信、达、雅** - 译文准确、表达得体、行文优雅
3. **体制内表达** - 使用规范公文语体
4. **涉敏审慎** - 敏感表述严格按口径处理
5. **时效标注** - 所有信息注明检索时间

---

## 📋 输出格式

### 新闻编译
- **标题**: [国别/机构] + 核心事件
- **正文**: 基本情况 / 主要内容 / 背景分析 / 研判建议

### 报告编译
- 呈报单位、信息来源、原文链接
- 内容摘要（200字以内）
- 核心要点（3-5条）
- 对华启示/参照意义
