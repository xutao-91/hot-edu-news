CEQ Issues Guidance on Categorical Exclusions
2026年4月9日，白宫环境质量委员会（CEQ）发布关于分类排除的新指南，指导联邦各部门和机构依据《国家环境政策法》（NEPA）建立、修订、采纳和应用分类排除程序。该指南旨在为基础设施项目减轻过度的许可审批负担，推进总统放松管制议程，降低建设成本，促进经济增长。根据NEPA规定，各机构可为通常不会产生重大影响的行为建立'分类排除'，当机构采取此类行动时，可采用简化方式履行NEPA义务，无需从头开始撰写分析报告。环境质量委员会主席Katherine Scarlett表示，关键基础设施项目的审批时间不应超过建设时间，联邦许可审批需要更高效运作。该指南考虑了2023年《财政责任法》对NEPA的修正案以及CEQ近期废止的NEPA实施条例。此外，CEQ还推出了技术工具简化分类排除的应用流程，包括'分类排除探索器'在线数据库和'CE Works'数字化平台。
