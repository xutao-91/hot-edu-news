美国教育部宣布针对参与被法院裁定非法的SAVE（学生贷款还款）计划借款人的后续措施。此前，SAVE计划因法律挑战被法院裁定超越教育部法定权限，该计划原本旨在为联邦学生贷款借款人提供更优惠的还款条件，包括降低月还款额和缩短还款期限。教育部表示将依法处理相关贷款，在确保借款人权益的同时遵守法院判决。对于已参与SAVE计划的约800万借款人，教育部将提供详细的过渡期安排，包括逐步恢复到之前的还款计划选项、提供个性化的还款计划转换指导、确保借款人在转换过程中不会受到惩罚性措施影响，如不会因转换而产生额外利息或费用。教育部还将通过官方网站、客户服务热线和合作服务机构向受影响借款人提供全面的信息支持，帮助他们了解可选的替代还款方案，包括标准还款计划、 graduated还款计划、延期还款选项以及其他收入驱动还款计划。教育部承诺将在整个过渡期间保持与借款人的沟通，确保他们能够顺利适应新的还款安排。
